Hugo操作

Jan 7, 2020
1 min read

  1. 查看hugo版本

    hugo version

  2. 新建博客页面

    新建博客页面myblog

    hugo new site myblog

  3. 下载主题

    下载主题m10c

    git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

  4. 新建博客

    在post目录下新建博客blog.md

    hugo new post/blog.md

  5. 本地启动

    hugo server -t m10c --buildDrafts

  6. 远程部署

    hugo --theme=m10c --baseUrl="https://youyuanbo.github.io/" --buildDrafts
